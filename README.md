This repository includes R scripts and data to recreate analyses for a manuscript entitled "Thermal pre-treatment of algal symbiont species differentially affects coral cell proliferation" currently in preparation.
In this manuscript we explore the effect of algal partner and thermal treatments on symbiont and host cell proliferation by offering Acropora tenuis larvae one of four algal species pre-exposed to elevated or ambient temperature. Additionally, we characterized the cell-surface glycome of each species-temperature combination to understand its role in symbiont recognition and proliferation. 

The "Cell-proliferation" directory includes scripts and data for analyzing symbiont colonization rates, symbiont density, and host cell division.
The "Sym-heat-stress" directory includes scripts and data for analyzing Fv/Fm and algal mortality.
The "Glycan" directory includes scripts and data for analyzing algal glycan abundance and composition.
